## 🇧🇷 Visão Geral
Sophie Bot é um assistente premium para servidores Discord que querem transformar a economia interna em uma experiência moderna de “banco digital”. Mesmo em pré-alpha, ela já combina painel bancário visual, sistema de investimentos futurista, score de cartão, comandos sociais e módulos de moderação — tudo com persistência segura em Postgres.

### Principais Recursos (pré-alpha)
- **Painel bancário interativo** (`/banco`) com embeds, botões e modais para depósitos, saques, transferências e boletos.
- **Investimentos avançados** (CDB, CDI, LCI/LCA, FIIs, cripto, índices) com rentabilidade simulada e boletins de mercado a cada 30 minutos.
- **Score de cartão de crédito** recalculado globalmente todos os dias, preparando o terreno para compras e limites variáveis.
- **Economia integrada**: trabalhar, daily, mineração, pesca, fazenda, loja, inventário, ranking e suporte multilíngue.
- **Infraestrutura robusta**: carregamento modular de comandos, logs Premium, rate limit configurável e persistência segura em Postgres.

### Tecnologias
- ![Node.js](https://img.shields.io/badge/Node.js-18+-green?logo=node.js&logoColor=white) ![Discord.js](https://img.shields.io/badge/Discord.js-14-5865F2?logo=discord&logoColor=white) ![Postgres](https://img.shields.io/badge/Postgres-dedicated-blue?logo=postgresql&logoColor=white)
- Persistência JSON centralizada na tabela `economy_state`, gerenciada pelo adaptador `PostgresJsonStorage`.

### Roadmap / Próximas iterações
- Configuração personalizada de canais para boletins e alertas.
- Empréstimos, PIX interno, seguros e planos premium.
- Compras com cartão de crédito, limites dinâmicos e cartões adicionais.
- Dashboard web para acompanhamento em tempo real.

## 🇺🇸 Overview *(Pre-alpha v0.0.1)*
Sophie Bot is a premium Discord banking experience for communities that want a futuristic, gamified economy. It ships with a fully interactive `/banco` command, advanced investments, future-ready credit scoring, and persistent Postgres storage — all wrapped in stylish embeds and modular code.

### Highlights (pre-alpha)
- **Immersive `/banco` UI** with deposits, withdrawals, transfers, invoices and actionable embeds/buttons.
- **Advanced investments** (fixed income, FIIs, crypto, indexes) with hourly projections and 30-minute market bulletins.
- **Credit score module (WIP)** with automatic daily refresh for every user.
- **Complete economy kit**: work, daily, mining, fishing, farming, shop, inventory, rankings, multi-language strings.
- **Hardened infrastructure**: modular command loader, premium logging, maintenance toggles, per-user rate limits, Postgres persistence.

### Tech Stack
- ![Node.js](https://img.shields.io/badge/Node.js-18+-green?logo=node.js&logoColor=white) ![Discord.js](https://img.shields.io/badge/Discord.js-14-5865F2?logo=discord&logoColor=white) ![Postgres](https://img.shields.io/badge/Postgres-dedicated-blue?logo=postgresql&logoColor=white)
- JSON persistence stored in the `economy_state` table through the `PostgresJsonStorage` adapter.

- Custom channel routing for alerts and daily bulletins.
- Lending, PIX-style transfers, insurance bundles.
- Credit card purchases, tiered limits, add-on cards.
- Web dashboard with live telemetry.
